Privacy Policy – Dexify

Last updated: August 10, 2025

Dexify does not collect, store, or share any personal information from its users. The extension only accesses publicly available profile data from GitHub and Reddit in order to display it in a Pokémon‑style card format.

Data Collection

No private or sensitive information is collected. Only public profile information (such as username, avatar, and public statistics) may be read from the page you are viewing.

Data Storage

Profile data may be temporarily stored locally using chrome.storage.local to improve performance. This data never leaves your device and is not transmitted to servers controlled by Dexify.

Third Parties

No data is sold, shared, or transferred to third parties.

Security

All processing happens locally within your browser. Dexify does not operate any backend that receives your data.

Contact

If you have questions about this policy, please contact: manfredi.rizza@gmail.com.

For transparency: this policy applies to the Dexify browser extension distributed via the Chrome Web Store.
